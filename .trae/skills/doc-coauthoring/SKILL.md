# Doc Co-Authoring Workflow

1.  **Context Gathering**: Ask about document type, audience, desired impact, and templates. Encourage info-dumping.
2.  **Refinement & Structure**: Iteratively build sections through brainstorming and editing.
3.  **Reader Testing**: Test the doc with a "fresh" perspective to catch blind spots.
