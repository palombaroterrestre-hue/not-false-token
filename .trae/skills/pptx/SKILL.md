# Presentation Design

1.  **Bold Palette**: Choose colors specific to the topic (avoid generic blue).
2.  **Visual Elements**: Every slide needs an image, chart, icon, or shape.
3.  **Layouts**: Use 2-column, grids, half-bleeds, and large stat callouts.
4.  **Motif**: Pick one distinctive element (e.g., rounded frames) and repeat it.
5.  **Typography**: Use interesting font pairings with personality.
