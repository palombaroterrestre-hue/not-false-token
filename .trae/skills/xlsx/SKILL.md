# Spreadsheet Standards

- **Formatting**: Use professional fonts (Arial) and standard color coding (Blue for inputs, Black for formulas).
- **Error-Free**: Ensure zero formula errors (#REF!, #DIV/0!, etc.).
- **Assumptions**: Keep all hardcoded assumptions in separate, documented cells.
- **Formulas**: ALWAYS use formulas instead of hardcoded values for calculations.
- **Analysis**: Use `pandas` for advanced data manipulation.
