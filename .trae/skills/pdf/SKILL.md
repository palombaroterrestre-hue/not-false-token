- **Reading**: Use `pypdf` or `pdfplumber` for text and table extraction.
- **Operations**: Merging, splitting, rotating, and metadata extraction.
- **OCR**: Use appropriate tools for scanned PDFs.
- **Form Filling**: Follow structured guidelines for interactive forms.
