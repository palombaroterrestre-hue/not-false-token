# Canvas Design Workflow

Follow these steps to create high-quality visual art:

1.  **Design Philosophy Creation (.md file)**: Define the movement (e.g., "Brutalist Joy") and its aesthetic rules (form, space, color, rhythm).
2.  **Visual Expression (.pdf or .png)**: Create the final artwork based on the philosophy. Focus on spatial communication, artistic interpretation, and minimal text.

## Essential Principles
- **Visual Philosophy**: Create an aesthetic worldview to be expressed through design.
- **Minimal Text**: Use sparse, essential text as a visual element, never lengthy blocks.
- **Spatial Expression**: Ideas communicate through space, form, color, and composition.
- **Expert Craftsmanship**: The final work must look meticulously crafted and professional.
