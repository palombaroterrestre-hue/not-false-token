# MCP Server Development

1.  **Research**: Understand API coverage vs. specialized workflow tools.
2.  **Infrastructure**: Set up TypeScript (recommended) with stdio or HTTP transport.
3.  **Tool Design**: Use descriptive names, clear input schemas (Zod), and actionable error messages.
4.  **Best Practices**: Prioritize comprehensive API coverage and concise data responses.
