# Frontend Design Principles

1.  **Bold Aesthetic**: Commit to a clear direction (e.g., brutally minimal, maximalist, retro-futuristic).
2.  **Typography**: Use distinctive, beautiful fonts (avoid generic Arial/Inter).
3.  **Color & Theme**: Use CSS variables and high-contrast palettes.
4.  **Motion**: High-impact animations and micro-interactions.
5.  **Spatial Composition**: Grid-breaking layouts, asymmetry, and generous negative space.
6.  **Visual Details**: Use textures, gradients, grain overlays, and noise to add depth.
